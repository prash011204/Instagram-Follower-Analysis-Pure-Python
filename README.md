# 📊 Instagram Follower Analysis using Pure Python

Analyze Instagram follower data using **Pure Python** without relying on external data analysis libraries such as Pandas or NumPy. This project demonstrates how fundamental Python concepts can be used to parse raw text data, organize information, perform analytical operations, and generate meaningful insights.

---

## 📌 Project Overview

This project focuses on extracting and analyzing Instagram follower information from a raw text dataset. The entire analysis pipeline has been implemented using core Python features including:

- File handling
- String manipulation
- Lists
- Dictionaries
- Loops
- Functions
- Sorting
- Searching
- Conditional statements

The primary objective of this project is to demonstrate strong Python programming skills by solving a real-world data analysis problem without using third-party libraries.

---

## 🎯 Objectives

- Read and process raw follower data from a text file
- Convert unstructured text into structured Python objects
- Perform statistical analysis using built-in Python functionality
- Analyze followers based on different profile attributes
- Implement searching and ranking algorithms
- Practice data parsing and data manipulation using Pure Python

---

## ✨ Features

- 📁 Read Instagram follower data from a text file
- 🔍 Parse raw text into structured records
- 👥 Count total followers
